> Sitio web dedicado a rankings, reseñas y comparativas de películas,
> donde la comunidad puede votar y comentar sus favoritas.

---

## 🌐 Demo en vivo
👉 [pelis-bm.vercel.app](https://pelis-bm.vercel.app)

## 📸 Vista previa
![Preview](assets/img/og-preview.jpg)

---

## ✨ Características
- 🎞️ Carrusel de películas destacadas
- 🏆 Ranking de las mejores y peores películas
- ⚖️ Comparativa entre películas
- 💬 Sección de comentarios
- 🔍 Barra de búsqueda
- 📱 Diseño responsive

---

## 🛠️ Tecnologías
- HTML5
- CSS3
- JavaScript (Vanilla)
- Vercel (deploy)

## 🚀 Instalación local
1. Clona el repositorio:
```bash
   git clone https://github.com/Kliderth/Pelis-BM.git
```
2. Abre `index.html` en tu navegador o usa Live Server en VS Code.
